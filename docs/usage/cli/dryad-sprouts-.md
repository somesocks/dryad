# dryad sprouts

```
$ dryad sprouts --help
dryad sprouts

Description:
    commands to work with dryad sprouts

Sub-commands:
    dryad sprouts run    execute sprouts in the garden
    dryad sprouts list   list sprouts in the current garden
    dryad sprouts path   return the path of the sprouts dir
```
