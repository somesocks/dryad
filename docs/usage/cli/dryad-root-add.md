# dryad root add

```
$ dryad root add --help
dryad root add <path> [alias]

Description:
    add a root as a dependency of the current root

Arguments:
    path    path to the root you want to add as a dependency
    alias   the alias to add the root under. if not specified, this defaults to the basename of the added root, optional
```
